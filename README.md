# Mystery Mansion Online

Учебная веб-версия детективной настольной игры в стиле Cluedo.

## Стек

- PHP 8+
- MySQL / MariaDB
- Vanilla JavaScript
- Canvas для игрового поля
- CSS-анимации

## Запуск

1. Скопируйте папку `cluedo_web` в `htdocs` или папку вашего локального PHP-сервера.
2. Создайте базу `cluedo_web`.
3. Импортируйте `sql/schema.sql`.
4. Проверьте доступ к MySQL в `includes/config.php`.
5. Откройте `http://localhost/cluedo_web/`.

По умолчанию:

```php
const DB_NAME = 'cluedo_web';
const DB_USER = 'root';
const DB_PASS = '';
```

## Что реализовано

- регистрация и вход;
- лобби матчей;
- создание игры;
- выбор персонажа/места;
- выход из ожидающего лобби;
- освобождение персонажа при выходе;
- автоудаление пустого ожидающего лобби;
- мультиплеер через AJAX-обновление;
- очередность ходов;
- бросок двух кубиков;
- Canvas-поле в виде плана особняка;
- комнаты, коридоры, двери и стартовые позиции;
- вариативность поля: разные матчи получают разные варианты дверей;
- проверка расстояния до комнат по реальным коридорам;
- запрет входа в комнату, если не хватает очков кубика;
- подсветка доступных клеток и комнат после броска;
- карты игроков;
- предложения версии;
- финальное обвинение;
- победы, поражения и винрейт;
- выезжающий детективный блокнот справа;
- журнал событий.

## Основная логика поля

Файл: `includes/data.php`

Ключевые функции:

- `mansion_rooms($gameId)` — комнаты, двери, секретные проходы и визуальные темы;
- `board_cells($gameId)` — данные поля для Canvas;
- `bfs_distances_from($x, $y, $gameId)` — расчёт расстояний по коридорам;
- `distance_to_target(...)` — проверка, можно ли дойти до клетки или комнаты;
- `reachable_targets(...)` — список доступных целей после броска кубиков.
